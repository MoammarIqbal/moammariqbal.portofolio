## Tentang Saya

Saya adalah Fullstack Web Developer yang berfokus pada pengembangan aplikasi web berbasis Laravel dan React. Berdasarkan pengalaman yang tercantum di CV, saya terbiasa membangun sistem terintegrasi, melakukan validasi data, mengembangkan fitur backend, serta menghubungkan sistem melalui API untuk kebutuhan operasional yang membutuhkan data konsisten dan real-time.

Pengalaman saya mencakup pengembangan sistem internal, dashboard, monitoring transaksi, dokumentasi API, serta pengelolaan dan visualisasi data. Saya pernah berkontribusi di Diskominfo Kabupaten Cirebon sebagai Fullstack Web Developer & Data Intern, kemudian melanjutkan pengalaman di Transferku sebagai Software Engineer Intern dan Staff mulai 1 Mei 2026.

Latar belakang Komputerisasi Akuntansi membantu saya memahami kebutuhan bisnis, pencatatan transaksi, laporan, dan pengolahan data. Kombinasi web development, API integration, data analytics, dan pemahaman proses bisnis menjadi nilai utama yang saya bawa dalam membangun solusi digital yang efisien, rapi, dan mudah digunakan.

### 1. AI-Powered Accounting Journal

![Landing Page AI-Powered Accounting Journal](assets/images/landingpage-akuntansi-nlp.png)

**Deskripsi:**  
AI-Powered Accounting Journal adalah sistem penjurnalan akuntansi yang terintegrasi dengan model AI. Sistem ini membantu pengguna non-akuntan mencatat transaksi menggunakan bahasa natural, kemudian mengolahnya menjadi jurnal hingga laporan seperti neraca saldo.

**Tujuan Project:**  
Mempermudah pencatatan akuntansi bagi pengguna yang tidak memiliki latar belakang akuntansi, sehingga proses input transaksi dapat dilakukan dengan kalimat sederhana dan lebih mudah dipahami.

**Role Saya:**  
Fullstack Developer

**Fitur Utama:**

- Input transaksi menggunakan bahasa natural.
- Bantuan AI untuk memahami transaksi dan menyusun jurnal.
- Jurnal umum.
- Chart of Account.
- Saldo awal.
- Buku besar.
- Neraca saldo.
- Aset tetap.
- Penyusutan.

**Teknologi yang Digunakan:**

- Laravel
- Blade
- Tailwind CSS
- MySQL
- Model AI / LLaMA 4

**Highlight Teknis:**

- Integrasi AI untuk menerjemahkan deskripsi transaksi menjadi pencatatan akuntansi.
- Alur akuntansi dari input transaksi sampai laporan dasar.
- Antarmuka yang disusun agar pengguna non-akuntan dapat memahami proses pencatatan.

**Hasil / Dampak:**

- Membantu menyederhanakan proses penjurnalan akuntansi.
- Mengurangi hambatan pengguna non-akuntan dalam memahami pencatatan debit dan kredit.
- Menjadi contoh penerapan AI pada domain akuntansi praktis.

### 3. AML Monitoring System

![Dashboard AML Monitoring System](assets/images/dasbor-aml.png)

**Deskripsi:**  
AML Monitoring System adalah sistem monitoring transaksi yang dikembangkan secara fullstack untuk PT. Graha Remitansi Indonesia (TransferKu). Sistem ini mendukung pemantauan transaksi untuk kebutuhan Anti-Money Laundering, fraud prevention, pencegahan pendanaan terorisme, dan risiko terkait senjata pemusnah massal.

**Tujuan Project:**  
Membantu perusahaan memonitor transaksi, melakukan screening sender dan beneficiary, mengklasifikasikan transaksi mencurigakan, serta menyediakan workflow review dan export laporan.

**Role Saya:**  
Fullstack Developer

**Fitur Utama:**

- Dashboard monitoring transaksi.
- Screening sender dan beneficiary menggunakan international sanction list.
- AML Engine untuk klasifikasi transaksi mencurigakan.
- Alert detail dan review kasus.
- Export laporan.
- Integrasi dengan payment gateway TransferKu.

**Teknologi yang Digunakan:**

- Laravel
- React
- Tailwind CSS
- MySQL
- API
- JavaScript

**Highlight Teknis:**

- Integrasi API untuk menerima dan memproses transaksi.
- Screening transaksi terhadap sanction list internasional.
- Dashboard real-time alerts untuk membantu proses monitoring.
- Workflow review dan export laporan untuk kebutuhan operasional compliance.
- Pemisahan frontend React dan backend Laravel untuk mendukung skalabilitas aplikasi.

**Hasil / Dampak:**

- Mendukung proses monitoring transaksi yang lebih terstruktur.
- Membantu identifikasi transaksi yang perlu ditinjau lebih lanjut.
- Menyediakan tampilan dashboard dan detail alert untuk mempercepat proses analisis.
